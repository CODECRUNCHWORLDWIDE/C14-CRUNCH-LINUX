# Week 1 — Exercises

Three exercises, ~30–40 min each. Type, don't paste.

1. **[Exercise 1 — The 50-command tour](exercise-01-fifty-commands.md)** — every command from Lecture 1, used in context.
2. **[Exercise 2 — Find the files](exercise-02-find-the-files.md)** — `find` puzzles.
3. **[Exercise 3 — Grep + pipes on a real log](exercise-03-grep-and-pipes.md)** — pipelining for fun and learning.

Run them in order. Each builds on the previous.

## Suggested workflow

- Open a fresh terminal.
- Open the exercise file alongside in your editor.
- Type each command yourself.
- Read the output.
- If you don't understand the output, **stop** and read the man page for the command before moving on.

The goal isn't to finish; the goal is muscle memory.
