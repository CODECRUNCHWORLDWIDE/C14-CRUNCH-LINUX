# Week 2 — Exercises

Three drills, ~5.5 hours total. Do them in order; each builds on the last.

| File | Time | Focus |
|------|------|-------|
| [exercise-01-awk-puzzles.md](./exercise-01-awk-puzzles.md) | 2h | Ten `awk` puzzles. Escalates from `print $1` to associative arrays. |
| [exercise-02-sed-substitutions.md](./exercise-02-sed-substitutions.md) | 1.5h | Ten `sed` substitutions. From `s/foo/bar/` to addressed ranges. |
| [exercise-03-real-log-pipeline.md](./exercise-03-real-log-pipeline.md) | 2h | Compose `awk` + `sed` + `grep` on real `/var/log` data. |

Commit your answers to your portfolio repo under `c14-week-02/exercises/`. Each puzzle should be its own short markdown file or a section in `answers.md`. Show the command **and** a sample of its output.

Where the prompt says "Ubuntu 24" or "Fedora 41," try it on whichever distro you have. If both — even better. Note any differences in your answers.

When in doubt, `man awk`, `man sed`, and `tldr awk` / `tldr sed`.
